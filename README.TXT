PROJECT TITLE: Game-Journey-Sun-Go-Kong-to-the-west-Java
PURPOSE OF PROJECT: The objective of this project is to create a game titled 'Game-Journey-Sun-Go-Kong-to-the-west-Java' using the Java programming language, with development taking place within Greenfoot. The goal of the game is to defeat the final enemy at level 3.
AUTHORS: Gede Agus Supriatmaja, Made Alit Juniska, Zulvan Avito Anwari, and Ni Ketut Rika Suryani.

note : 
Game-Journey-Sun-Go-Kong-to-the-west-Java is a game project created using the Java programming language, developed within the Greenfoot environment. The goal of this game is to assist Sun Go Kong in his journey to the West to defeat the final king. There are 3 levels before encountering the ultimate king: easy, medium, and difficult stages.
